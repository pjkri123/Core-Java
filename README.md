# 🌟 Core Java – Practice & Revision Repository

This repository contains all the **Core Java examples and practice programs** I have written to **revise concepts**, strengthen fundamentals, and prepare for real-world Java development and interviews.

It includes hands-on programs covering OOP, Collections, File Handling, Annotations, Reflection, and much more.

---

## 📁 Folder Structure

| Folder Name        | Description |
|--------------------|-------------|
| `annotation`       | Practice programs related to built-in and custom Java annotations. |
| `collections`      | Examples using List, Set, Map, Queue, and other Collection Framework utilities. |
| `example`          | Basic Java examples for revising syntax and fundamental concepts. |
| `fileHandling`     | Programs for reading, writing, buffering, and file manipulation in Java. |
| `immutableClass`   | Practice on creating immutable classes and understanding immutability. |
| `javadoc`          | Code documenting how to generate Javadoc for Java classes and methods. |
| `objectClass`      | Examples revising `equals()`, `hashCode()`, `toString()`, cloning, etc. |
| `reflection`       | Practice programs using Java Reflection API to inspect classes at runtime. |

---

## 🎯 Purpose of This Repository

- To revise and strengthen **Core Java concepts**  
- To practice fundamental and advanced topics in Java  
- To prepare for **interviews**, coding tests, and backend development roles  
- To maintain organized notes and examples for future learning  
- To improve confidence in Java 17 and OOPs fundamentals
